# nfc.removeListener()

> --------------------- ------------------------------------------------------------------------------------------
> __Type__              [Function](https://docs.coronalabs.com/api/type/Function.html)

> __Return value__      none

> __See also__          [nfc.*](/plugin/nfc/)
> --------------------- ------------------------------------------------------------------------------------------

## Overview

Removes a previously defined with [nfc.setListener()](/plugin/nfc/setListener) listener.

## Syntax
```lua
nfc.removeListener()
```
