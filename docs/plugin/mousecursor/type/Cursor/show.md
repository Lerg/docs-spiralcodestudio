# object:show()

> --------------------- ------------------------------------------------------------------------------------------
> __Type__              [Function](https://docs.coronalabs.com/api/type/Function.html)

> __Return value__      none

> __See also__          [mousecursor.*](/plugin/mousecursor/)
> --------------------- ------------------------------------------------------------------------------------------

## Overview

Makes a previously hidden or just created cursor visible.

## Syntax
```lua
object:show()
```
